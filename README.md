CronUI
======

CronUI is an web-based tool based on Python Flask, jquery and bootstrap to edit, delete and create cron jobs.

## Features

* list cron jobs
* modify cron jobs
* delete cron jobs
* create cron jobs

## Requirements

* Python
* Flask
* Flask-JSGlue

## Installation

* pip install Flask
* pip install Flask-JSGlue

* git clone ...

* cd CronUI
* export FLASK_APP=index.py
